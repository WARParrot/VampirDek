# Example Blood Cult mod

This is the modding proof slice. It adds three cards, one opponent deck, one encounter, localization placeholders, and a reward pool. Runtime bootstrap spawns an EncounterPoint in ExplorationMode when the encounter is loaded, making the duel reachable from exploration via the normal encounter trigger.

Current supported JSON folders: cards, enchantments, decks, encounters, layouts, phasegraphs, winconditions, hints.
